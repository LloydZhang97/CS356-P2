This is the minitwitter project for CS356

To run the project: javac Driver.java, java Driver
There may be warnings thrown, ignore them.

The use of admin manager is pretty self explanatory except for one thing
To add a user/group, select the group to add the user/group in in the tree view on the left, then add the desired group/user
ex. If I wanted to add Jill into the root group, I would click on the "root", type Jill into the field, and then click "Add User"
